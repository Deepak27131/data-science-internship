# Titanic ML Project

This repository contains a Python machine learning project that explores the Titanic dataset, builds predictive models, and evaluates their performance. It is structured for learning and experimentation with data preprocessing, feature engineering, model training, and evaluation.

---

## 📌 Project Overview

- **Goal:** Predict passenger survival on the Titanic using tabular data and machine learning techniques.
- **Dataset:** `data/titanic.csv` (standard Kaggle Titanic dataset format).
- **Key components:** data preprocessing, model training, evaluation, and exploratory analysis in Jupyter notebooks.

---

## 🚀 Getting Started

### 1) Install dependencies

This project uses a `requirements.txt` file to describe Python dependencies.

```bash
cd "d:\Data Sience 6 month internship\titanic_ml_project"
python -m pip install -r requirements.txt
```

### 2) Run the main pipeline

The main entrypoint is `main.py`.

```bash
python main.py
```

> ✅ `main.py` typically orchestrates data loading, preprocessing, model training, evaluation, and output generation.

---

## 🗂️ Repository Structure

- `data/` - Contains the Titanic dataset (`titanic.csv`).
- `src/` - Python modules for preprocessing, model training, evaluation, and utilities.
- `notebooks/` - Jupyter notebooks for exploratory analysis and experiments.
- `reports/` - Markdown or output reports summarizing findings.
- `main.py` - Primary script to run the pipeline.
- `requirements.txt` - Python dependencies.
